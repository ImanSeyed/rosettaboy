RosettaBoy Python
=================
My first ever attempt at an emulator

Usage
-----
```
python3 -m venv venv
./venv/bin/pip install pygame
./venv/bin/python3 main.py game.gb
```

Requirements
------------
- Python 3.6+
- PyGame

Formatting
----------
Automated with Black:
```
./venv/bin/black src/*.py
```
